## Critical (2)

| # | Domain | Issue | Location |
|---|--------|-------|----------|
| 1 | Security | **Weak NEXTAUTH_SECRET** — human-guessable string allows JWT forgery | `.env:2` |
| 2 | Accessibility | **No skip link or `<main>` landmark** — keyboard users cannot bypass navigation | `app/(root)/layout.tsx` |

### Details

**1. Weak NEXTAUTH_SECRET**
- **Standard:** OWASP A02:2021 – Cryptographic Failures
- **Evidence:** `NEXTAUTH_SECRET=example-insecure-secret-value`
- **Impact:** Attacker who guesses this can forge session tokens, impersonate any user.
- **Fix:** Generate with `openssl rand -base64 48`. Rotate immediately.

**2. No Skip Link or Main Landmark**
- **Standard:** WCAG 2.4.1 Bypass Blocks (A)
- **Evidence:** `app/(root)/layout.tsx` wraps content in `<div>`, no skip link, no `<main>`.
- **Impact:** Keyboard/screen reader users must tab through entire Header on every page.
- **Fix:** Add `<a href="#main-content" className="sr-only focus:not-sr-only">Skip to main content</a>` and wrap children in `<main id="main-content">`.

### Details

**4. No Security Headers**
- **Standard:** OWASP A05:2021 – Security Misconfiguration
- **Evidence:** `next.config.ts` has no `headers()` configuration.
- **Impact:** Missing CSP allows injected scripts; missing HSTS allows SSL stripdown.
- **Fix:** Add `headers()` in `next.config.ts` returning CSP, HSTS, X-Frame-Options, X-Content-Type-Options.
