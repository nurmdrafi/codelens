# Codelens v0.0.4 Tool-Substitution Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Substitute Biome (JS/TS quality+a11y) and knip (JS/TS dead-code) for the corresponding rg pattern blocks in codelens-reviewer, with rg fallback when the tools aren't installed. Ship as v0.0.4.

**Architecture:** Tool substitution, not prompt micro-optimization. Phase 2 probes for Biome+knip at runtime; present → use them via `ctx_execute` (output auto-indexes); missing → fall back to existing rg patterns. Phase 0 drops upfront preflight for graceful degradation. Phase 4 extracts the report template. Language scope declared explicit (JS/TS only).

**Tech Stack:** Markdown (agent body, skill, references). Bash + npm-installed tools (Biome 2.5.0, knip 6.17.1). context-mode MCP + Context7 MCP + ripgrep (existing codelens deps).

**Spec:** `docs/superpowers/specs/2026-06-16-codelens-v0.0.4-overhead-optimization-design.md`
